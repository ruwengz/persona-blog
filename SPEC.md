# 个人网站项目规范 - 机甲风格博客

## 1. 项目概述

- **项目名称**: 机甲风格个人博客
- **项目类型**: Hexo静态博客网站
- **核心功能**: 展示个人文章、作品和互动交流的博客平台
- **目标用户**: 技术博主、游戏爱好者、机甲/科幻文化爱好者

## 2. 视觉与设计规范

### 色彩方案
- **主色调**: #0a192f (深空蓝/机甲暗色)
- **次要色**: #1a1a2e (暗紫色)
- **强调色**: #00ff88 (霓虹绿/机甲指示灯色)
- **辅助强调色**: #ff0055 (警告红)
- **文字色**: #e6f1ff (亮白)
- **次要文字色**: #8892b0 (灰蓝)

### 字体规范
- **标题字体**: 'Orbitron', 'Rajdhani', sans-serif (科幻机甲感)
- **正文字体**: 'Noto Sans SC', 'Microsoft YaHei', sans-serif
- **代码字体**: 'Fira Code', 'Consolas', monospace

### 视觉效果
- 霓虹发光边框效果
- 扫描线动画
- 粒子/网格背景
- 按钮悬停时的机甲变形效果
- 页面切换时的渐入渐出效果

### 设计元素
- 六边形、几何线条装饰
- 扫描线纹理
- 发光边框和阴影
- 图标使用线性风格

## 3. 页面结构

### 页面列表
1. **首页 (/)**: 展示精选文章、作者介绍、社交链接
2. **日常 (/daily/)**: 生活随笔、日常分享
3. **专题 (/topics/)**: 专题系列文章
4. **留言板 (/message/)**: 访客留言互动
5. **关于 (/about/)**: 关于博主介绍

### 导航结构
- 顶部导航栏（带霓虹灯效果）
- 首页 > 日常 > 专题 > 留言 > 关于

### 响应式设计
- 桌面端: 完整布局
- 平板端: 简化导航
- 移动端: 汉堡菜单

## 4. 功能特性

### 核心功能
- 文章发布与管理（Markdown支持）
- 分类和标签系统
- 留言板功能
- 搜索功能
- RSS订阅
- 归档功能

### 交互设计
- 平滑滚动效果
- 导航栏悬停高亮
- 文章卡片悬停效果
- 回到顶部按钮
- 加载动画

## 5. 技术选型

### 框架与工具
- **博客框架**: Hexo 7.x
- **主题**: hexo-theme-butterfly (机甲风格魔改)
- **Node.js**: 18.x+
- **包管理**: npm

### 主题配置
- 使用 hexo-theme-butterfly 配合机甲风格魔改
- 安装必要插件: hexo-word-counter, hexo-generator-search

### 日常页面
